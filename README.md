# Hotel

Sistema de Control de Inventarios hotelero.

## Descripción 
Este repositorio es un punto de partida para los  hoteles en su sistematisacion y avance en la recopilacion de la informacion que se estructura del dia a dia que necesita ser guaradada para un mejor control y funcionamiento,  los derechos de autor estan resrervados por  el desarrollador del sotfware diseñado para el Hotel Del Mar.
### Requisitos
Para su correcto funcionamiento se requiere tener instaladas las siguientes herramientas:

1: pagar un hostin que permita que una url pueda almacenar  el sitio de partida de un puerto, permitiendo acceder desde cualquier ordenador con la direcion url.
